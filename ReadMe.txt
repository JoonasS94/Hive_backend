Miten pyöritetään serveriä:
Asennetaan ensin venv.
1. Avaa cmd kansiossa GitHub\Hive_backend
2. python -m venv venv
3. venv\Scripts\activate
4. pip install -r requirements.txt
5. python manage.py runserver
6. 127.0.0.1:8000/admin/ tunnus: Admin salasana: root
7. Endpointit voit tarkastella swaggerilla osoitteessa 127.0.0.1:8000/swagger/
